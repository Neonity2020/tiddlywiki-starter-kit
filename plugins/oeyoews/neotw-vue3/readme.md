## For developers

```js
const vue = require('$:/plugins/oeyoews/neotw-vue3/vue.global.prod.js')
```

* 使用示例参照 $:/plugins/oeyoews/neotw-vue3/widget.js

* 更多用法请参考 [Vue3 官方文档](https://cn.vuejs.org/guide/essentials/application.html)
